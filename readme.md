# 🔐 SecureReceipt – Blockchain-Based E-Receipt Verification System

SecureReceipt is an innovative platform that ensures the **authenticity** and **integrity** of digital purchase receipts using **blockchain** and **RSA-based digital signature verification**. Built for Walmart Sparkathon, the system guarantees tamper-proof receipts while offering a seamless UI for scanning, uploading, and verifying e-receipts.

## 🚀 Features

- ✅ **Blockchain-Powered Receipt Ledger**
- 🔏 **RSA Digital Signature Verification**
- 📦 **Tamper-Proof Storage of E-Receipts**
- 📄 **QR Code Receipt Upload & Scan**
- 🌐 **Modern Frontend UI (Vite + React + TailwindCSS)**
- 📊 **Detailed Verification Dashboard**
- ⚡ Fast & Lightweight Performance
